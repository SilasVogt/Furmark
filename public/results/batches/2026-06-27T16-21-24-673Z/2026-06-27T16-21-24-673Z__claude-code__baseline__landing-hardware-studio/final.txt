## What changed

A complete premium landing page for the **Arcwell** modular desk lamp, replacing the scaffold. Files:

- **`src/main.tsx`** — the full page: sticky nav, hero "studio", spec sheet, modular-system section, closing pre-order band, footer. Configuration state (arm / lens / finish) lifts to `App` and drives the lamp, the live price, the pick labels, and the closing summary.
- **`src/lamp.tsx`** — a **generated SVG product visual**. The lamp is drawn with forward-kinematics so the three modules actually change it:
  - **Arm** (Pivot / Boom / Compact) → recomputes the articulated pose; the beam always lands on the desk.
  - **Lens** (Spot / Flood / Diffuse) → beam width, intensity, and glow temperature (warm 3000K → cool 4000K).
  - **Finish** (Graphite / Aluminium / Midnight / Oxide) → cross-fades the metal color with a sheen overlay.
  - Plus compact glyphs for the modular-system lineup.
- **`src/data.ts`** — typed config model, price math, and the spec table.
- **`src/style.css`** — dark cinematic "studio" system: near-black palette with a single light glow that tracks the active lens, Space Grotesk / Inter / Space Mono (graceful system fallback), accessible toggle controls, and responsive grids with overflow guards.
- **`index.html`** — title, meta description, theme-color, font preconnects.

Design intent per the brief: the first viewport *is* the product — a glowing lamp on a stage with the configurator beside it — and the palette deliberately avoids beige/brass (graphite, brushed aluminium, midnight lacquer, oxide). Controls are labeled `01 — Arm / 02 — Lens / 03 — Finish` with `role="group"`, `aria-pressed`, and per-option `aria-label`.

## Verification

- **`pnpm build`** (`tsc --noEmit && vite build`) — passes clean.
- **Headless Chromium screenshots** at 1440 / 980 / 768 / 390 px — first viewport clearly shows the lit lamp, no text overlap, no horizontal overflow, and the breakpoint transition (two-column ↔ one-column) is clean.
- **Interactivity check** — temporarily defaulting to Boom + Flood + Aluminium confirmed all three dimensions re-render the SVG (pose, beam spread/color, metal finish), with active states, labels, and price updating; then reverted to defaults.

One note: fonts load from Google Fonts and fall back to the system stack if offline (the build and layout don't depend on them). The untracked `dist/` is just the build output.
